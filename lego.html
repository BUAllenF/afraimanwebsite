Lego Smart Clock

Introducing my homemade lego smart clock. Using a real time clock, temperature/humidity sensor, and an Arduino it can be used to read the current date/time, set alarms, and read the temperature/humidity in the environment.

Materials

  • Arduino UNO
  • DS1307 Real Time Clock
  • DHT22 Temperature/Humidity Sensor
  • LCD Screen
  • Legos

Skills Used

  • Arduino IDE
  • Circuit Design
  • Soldering
  • Engineering Design
